# Concept Museum Exhibition Schedule

A single-file, Swiss/International Typographic Style concept for a museum exhibition schedule.

## What’s in this repo
- `swiss-schedule.html`: the main experience (single file).
- `index.html`: redirect/entrypoint for GitHub Pages.

## Safety / Privacy
- No API keys or secrets are used.
- No external network calls are required to run the page.

## Run locally

```bash
python3 -m http.server 5564 --bind 127.0.0.1
```

Then open `http://127.0.0.1:5564/`.

## Deploy to GitHub Pages
1. Push to the `main` branch.
2. In GitHub: **Settings → Pages**.
3. Set **Build and deployment** to **Deploy from a branch**.
4. Choose **Branch: main** and **Folder: /** (root).
